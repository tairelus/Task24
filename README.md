# Task24
Разберись как в Java запускать, приостанавливать и прерывать потоки.
Напиши для чего используются методы:  interrupt, sleep, join
Как узнать имя текущего потока?
Как узнать жив ли поток?
Когда в потоке возникает InterruptedException ?

Реализуй класс который можно использовать в потоках (можешь назвать его MessageLoop)
Создай в классе MessageLoop массив из трех строк (можешь назвать его messagesArray).
Реализуй в классе MessageLoop метод run
Метод run должен выводить в консоль элементы массива messagesArray по одному с интервалом 4 секунды.

Напиши многопоточную программу которая будет состоять из двух потоков:
Первый поток это основной поток (main thread) - должен запускать второй поток.
Второй поток - должен использовать класс MessageLoop
Перед стартом второго потока  первый поток должен выводить уведомление: ”Старт MessageLoop thread”
После старта второго потока первый поток должен выводить уведомление:”Жду пока второй поток закончит свое исполнение”
Пока второй поток жив основной поток должен выводить в консоль уведомление каждую секунду: "Я жду..."
После завершения второго потока первый поток должен вывести
уведомление:”Конец!”
Строка вывода программы должна быть в формате:
имя потока: уведомление
